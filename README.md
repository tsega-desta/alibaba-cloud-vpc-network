# Alibaba Cloud VPC Network Architecture

This project demonstrates a secure enterprise network architecture built using Alibaba Cloud networking services.

## Architecture Overview

The design includes public and private subnets with secure connectivity.

Components:

- Virtual Private Cloud (VPC)
- NAT Gateway
- Elastic IP
- Security Groups
- Linux servers

## Features

• Secure cloud networking  
• Public and private subnet separation  
• Controlled internet access using NAT Gateway  
• Security group firewall rules

## Technologies

- Alibaba Cloud VPC
- NAT Gateway
- Elastic IP
- Linux networking
